Das [[Context-Window sollte Aufgabenbezogen bleiben\|Context-Window sollte Aufgabenbezogen bleiben]] eines Agents sollte immer auf eine einzelne Aufgabe bezogen sein. Auf diese Weise soll die Gefahr der [[Halluzination\|Halluzination]] verringert werden, da das Modell nicht mehr zwei verschiedene Themen “vermischen” kann.
