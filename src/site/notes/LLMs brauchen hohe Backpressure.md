Eine [[LLM]] funktioniert am Besten, wenn sie ausreichend [[Backpressure\|Backpressure]] hat, soll heißen: zu keinem Zeitpunkt auf verzögerten externen (meist menschlichen) Input warten muss. Hierzu ist es am Besten, automatisches [[Testing\|Testing]] aufzusetzen – so kann die Iterationsgeschwindigkeit erhöht werden, während Korrektheit sichergestellt werden kann.
