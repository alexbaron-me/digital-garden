> ReAct = **Re**ason & **Act**

Eine Methode für [[Harness]]es, welche es einem Modell ermöglicht, bei Bedarf in [[Reasoning\|Reasoning]] zu wechseln (ähnlich zum Planning Mode z.B. bei Claude Code). Anschließend werden Aktionen ausgeführt, und die Ergebnisse wieder in das Reasoning gegeben. So soll der [[Agent\|Agent]] bessere Entscheidungen treffen, und sinnvollere Aktionen durchführen.
